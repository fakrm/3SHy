System Overview

This Python-based web server implements a secure user authentication system with database integration. It handles user registration with email verification, login validation, and serves an energy data dashboard interface. Built using Python's http.server module and MySQL for database management, the system provides a complete solution for secure access to energy consumption data.

The server implements a two-step registration process for enhanced security. When users sign up, their information isn't immediately stored in the database. Instead, a unique verification token is generated and an email with a verification link is sent to the provided address. Only after clicking this link (which confirms ownership of the email) is the user's information added to the database with a verified status. This approach significantly reduces the risk of fake accounts and ensures data integrity.

Technical Implementation

The server handles different types of HTTP requests through its specialized methods:

  • POST requests: Handle form submissions for signup and login processes
  • GET requests: Manage page navigation, resource loading, and email verification

The verification system stores tokens in memory (which would be replaced by database storage in a production environment) with a 24-hour expiration period. This provides a reasonable timeframe for users to complete verification while maintaining security.

Upon successful login, the server queries all database tables and formats the data into a JavaScript object that powers the dashboard interface. This allows users to interact with energy data through tables and charts, filtering by date and searching for specific information.

Dashboard Functionality

Once logged in, users can access the Energy Data Dashboard that provides multiple ways to analyze energy consumption data:

  • 15-Minute Intervals: View granular energy usage patterns
  • Daily Data: Examine consumption trends by day
  • Average By Date: Analyze average consumption over time periods

The dashboard includes search functionality and date filters to help users focus on specific time periods or data points of interest. All information is presented both in tabular format and through interactive charts for improved data visualization.
